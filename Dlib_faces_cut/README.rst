Face Cropper
############

Introduction
************

#. 將檢測到的人臉裁切下來, 依次排序顯示在新的圖像上;

   .. code-block:: bash

      python3 crop_faces_show.py


#. 將檢測到的人臉生成單個圖像儲存到本地路徑;

   .. code-block:: bash

      python3 crop_faces_save.py


#. 將檢測到的人臉,依次在攝影頭窗口即時顯示;

   .. code-block:: bash

      python3 faces_from_camera.py
