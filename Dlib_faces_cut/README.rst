Face Cropper
############

Introduction
************

#. 将检测到的人脸剪切下来, 依次排序平铺显示在新的图像上;

   .. code-block:: bash

      python3 crop_faces_show.py


#. 将检测到的人脸生成单个图像存储到本地路径;

   .. code-block:: bash

      python3 crop_faces_save.py


#. 将检测到的人脸,依次在摄像头窗口实时平铺显示;

   .. code-block:: bash

      python3 faces_from_camera.py
